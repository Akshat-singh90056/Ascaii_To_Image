# ASCII Image Converter (Terminal-Based)

Convert normal images into ASCII art directly in the terminal.

---

## Features

- Converts any image (PNG, JPG, JPEG) into ASCII format.
- Adjustable resolution / scale factor.
- Optionally saves ASCII output to a text file.
- Simple CLI usage.
